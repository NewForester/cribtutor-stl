Boost

Boost is a large and diverse collection of open source libraries that extend the STL. It is not part of the STL and is beyond the scope of this tutorial.

Boost Libraries

Here are listed a number of Boost libraries that may well have been included in C++ 11.

  1. The array<> class template is a fixed size array that meets most of the requirements for a sequence container.
  2. The dynamic_bitset<> class template is similar to the STL bitset<> but its size can change at run-time.
  3. The binder library generalises the STL bind1st<> and bind2nd<> function templates and adds place-holders.
  4. The compose library allows multiple functors to be combined into a single functor for use with STL algorithms.
  5. Boost generalises the STL functor adaptors 'mem_fun' etc.
  6. The function library provides a generic function wrapper library.
  7. There is an alternative <functional> header file that avoids the reference-to-a-reference issue.
  8. There is a lamba function library that provides an alternative to the use of binders, composers and adaptors and offers features of functional programming.
  9. There are several smart pointer libraries that provide solutions to a number of problems with the STL auto_ptr<>.
