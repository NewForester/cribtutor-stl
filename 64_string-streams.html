STL Streams

String Streams

String streams provide for formatted input and output to and from buffers of type std::string.

String streams are implemented by the classes istringstream, ostringstream and stringstream in the header file <sstream>.

The string stream specific methods are:

  1. istringstream(mode=ios::in) is the default constructor allowing extraction.
  2. ostringstream(mode=ios::out) is the default constructor allowing insertion.
  3. stringstream(mode=ios::out|ios::in) is the default constructor allowing extraction and insertion.
  4. istringstream(str,mode=ios::in) initialises the stream object with a copy of 'str'.
  5. ostringstream(str,mode=ios::out) initialises the stream object with a copy of 'str'.
  6. stringstream(str,mode=ios::out|ios::in) initialises the stream object with a copy of 'str'.
  7. str() gets the contents of the stream object.
  8. str(str) sets the contents of the stream object to a copy of 'str'.
