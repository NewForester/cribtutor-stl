STL Introduction

STL stands for Standard Template Library. The STL is to C++ as the standard library is to C.

The use of templates ensures the STL is generic enough to work with application specific classes.

As a consequence the syntax is not easy to read until you get used to it. This set of crib sheets concentrates on semantics and avoids using too much nasty syntax.

The STL is large. A tutorial is no substitute for practical experience but these crib sheets and the cribtutor program will, it is hoped, not only help you learn what the STL comprises but also retain and recall what you have learnt effectively.

This tutorial does not cover C++ 11.

Good luck and emjoy !
