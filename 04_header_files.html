C Header Files

The original C++ 98 standard defines 18 header files that correspond to the C 94 standard. These are not covered in this tutorial.

All are provided for reasons of C compatibility and have not been superseded although most are little used.

  1. <cassert> - assert macro
  2. <cctype> - (narrow) character type macros (tolower() and friends)
  3. <cerrno> - errno macro
  4. <cfloat> - limits for floating point types
  5. <ciso646> - empty
  6. <climits> - limits for integer types
  7. <clocale> - localisation functions
  8. <cmath> - mathematical functions (sin() and friends)
  9. <csetjmp> - setjmp() and longjmp()
  10. <csignal> - signal handling
  11. <cstdarg> - handling of variable length argument lists
  12. <cstddef> - defines NULL
  13. <cstdio> - C standard i/o (printf() and friends)
  14. <cstdlib> - C standard library functions (malloc() and friends)
  15. <cstring> - (narrow) character string handling (strcpy() and friends)
  16. <ctime> - date and time functions
  17. <cwchar> - wide character string handling
  18. <cwctype> - wide character type macros
